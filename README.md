# Dot files

## Pre-requisites

1. zsh
2. oh-my-zsh
3. starship
4. neovim
5. Latest Nerd Font of your choosing

## Setup

1. Run `scripts/create-symlinks.sh`.
2. Run `scripts/create-zsh-functions.sh`.
